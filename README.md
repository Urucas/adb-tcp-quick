# adb-tcp-quick
Quickly connect an android device to adb over wifi

#Install
```bash
$ npm install -g adb-tcp-quick
```

#Usage
Use ```adb-tcp-quick``` to get a list of devices to connect via tcp 
or ```adb-tcp-quick -d``` to disconnect a device.
